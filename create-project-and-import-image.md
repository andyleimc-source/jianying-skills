# Skill: 创建项目并导入本地图片

## 描述
在剪映专业版中创建一个新项目，从本地下载文件夹导入图片素材，并保存项目。

## 适用场景
- 快速开始一个新的视频/图片剪辑项目
- 将本地图片导入剪映进行后续编辑

## 关键步骤

### 1. 打开剪映专业版
```
Bundle ID: com.lemon.lvpro
应用路径: /Applications/VideoFusion-macOS.app
```

### 2. 创建新项目
- 快捷键：`Cmd + N`
- 或点击主页的「开始创作」按钮
- 项目默认以当天日期命名（如"3月26日"）

### 3. 导入本地图片
- 快捷键：`Cmd + I` 打开文件选择器
- 在路径输入框使用 `Cmd + Shift + G` 跳转到指定目录
- 输入图片完整路径，如：`/Users/andy/Downloads/IMG_9467.PNG`
- 按 Return 确认选择
- 再按 Return 完成导入

### 4. 保存项目
- 项目会自动保存到本地
- 也可用 `Cmd + S` 手动保存
- 默认保存位置：`/Users/andy/Movies/JianyingPro/User Data/Projects/com.lveditor.draft/`

## 注意事项
- 剪映专业版支持的图片格式：JPG、JPEG、PNG、HEIC、WEBP 等
- 项目自动保存间隔约 1 分钟
- 文件选择对话框需要用键盘操作（Cmd+Shift+G 输入路径）

## 素材导入路径参考
```
下载文件夹: ~/Downloads/
图片文件夹: ~/Pictures/
桌面:       ~/Desktop/
```

## 相关快捷键汇总
| 操作 | 快捷键 |
|------|--------|
| 新建项目 | Cmd + N |
| 导入素材 | Cmd + I |
| 保存项目 | Cmd + S |
| 路径跳转 | Cmd + Shift + G |
