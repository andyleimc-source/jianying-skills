# 剪映 Skills 合集

> 收集并管理在 macOS 本地剪映专业版（CapCut for Mac）中常用操作的 Skill 文档，方便随时查阅和复用。

## 什么是 Skill？

每个 Skill 是一份操作指南，记录在剪映中完成某类任务的关键步骤、快捷键和注意事项。由 Claude Code AI 辅助整理生成。

## Skill 列表

| Skill | 描述 |
|-------|------|
| [创建项目并导入本地图片](./create-project-and-import-image.md) | 新建剪映项目，从本地文件夹导入图片素材并保存 |

## 环境信息

- **应用**: 剪映专业版 (VideoFusion-macOS)
- **Bundle ID**: `com.lemon.lvpro`
- **平台**: macOS
- **默认项目保存路径**: `~/Movies/JianyingPro/User Data/Projects/`
